# ALLMYNE × Miles & More — Mobile‑First Landing (React + TS + Vite + Tailwind)

Bản **mobile-first** với navbar hamburger, layout cột dọc trên mobile, và mở rộng lên grid ở `md:`.

## Chạy dự án
```bash
npm install
npm run dev
# mở http://localhost:5173
```

## Tùy chỉnh nhanh
- Ảnh: `src/assets/images.ts`
- Màu thương hiệu: `tailwind.config.js` (mục `colors.brand`)
- Logo: thay `src/assets/logo.svg`
- Các section: `src/components/*`
